# 博客

一个使用 Vue3 和 Rust 构建的练习博客项目。

## 项目结构

-   **frontend**: 使用 Vue 3、TypeScript、Naive UI 和 Tailwind CSS 构建的前端应用
-   **backend**: 使用 Rust (Axum) 和 SQLite3 构建的后端 API

## 技术栈

### 前端

-   Vue 3
-   TypeScript
-   Naive UI
-   Tailwind CSS

### 后端

-   Rust (Axum 框架)
-   SQLite3

## 运行项目

### 前端

```bash
cd frontend
npm install
npm run dev
```

### 后端

```bash
cd backend
cargo run (开发环境)
.\target\release\backend.exe (生产环境)
```

## 说明

这是一个用于学习目的的练习项目。不打算部署到生产环境，有建议欢迎探讨。
